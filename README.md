# neo4j-manager
mange multiple neo4j databases

# usage
```
usage: neo4j-manager [-h] [--verbose] {ls,add,activate,delete} ...

positional arguments:
  {ls,add,activate,delete}
    ls                  list all available databases
    add                 add a new neo4j database
    activate            activate a different available neo4j database
    delete              delete a neo4j database

optional arguments:
  -h, --help            show this help message and exit
  --verbose             print all informational messages
  ```
